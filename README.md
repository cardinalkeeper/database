Схема базы данных Кардинал Кипера
===================================

Использование скрипта
---------------------

Инсталяция базы данных.

```
index.js install 
```

Обновление базы данных до последней миграции.

```
index.js update
```
```

Получить версию последней миграции.

```
index.js version
```
